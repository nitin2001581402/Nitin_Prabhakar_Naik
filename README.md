Java solutions for the Tandemloop Full Stack Developer Screening Test.

## Language
- Java (JDK 17)

---

## Problems

### Problem 1: Calculator
- **File:** Problem1.java  
- **Description:** Performs addition, subtraction, multiplication, and division using a class.  
- **Input:** Two numbers (`a`, `b`) and operation type (`add`, `subtract`, `multiply`, `divide`)  
- **Output:** Result of the selected operation  

### Problem 2: Odd Number Series
- **File:** Problem2.java  
- **Description:** Generates a series of odd numbers up to the given input.  
- **Input:** Single integer  
- **Output:** Series of odd numbers (1, 3, 5, …)  

### Problem 3: Conditional Odd Number Series
- **File:** Problem3.java  
- **Description:** Generates a conditional odd number series based on input.  
- **Input:** Single integer  
- **Output:** Series pattern as per the examples  

### Problem 4: Count Multiples
- **File:** Problem4.java  
- **Description:** Counts numbers in a list that are multiples of 1 to 9.  
- **Input:** List of integers  
- **Output:** Map showing counts for each number 1–9

## How to Run

1. Open Terminal / Command Prompt in the project folder.  

2. **Compile the programs:**
   ```bash
   javac Problem1.java
   javac Problem2.java
   javac Problem3.java
   javac Problem4.java
   
3. Run the programs:
   ```bash
   java Problem1   # Calculator
   java Problem2   # Odd Number Series
   java Problem3   # Conditional Odd Number Series
   java Problem4   # Count Multiples
